# Coin Flip Game 

A simple Coin Flip Game built using **HTML5** and **JavaScript**. The user chooses **Heads** or **Tails**, and the computer randomly flips a virtual coin using `Math.random()`. The game then displays whether the user wins or loses.

## Features

* Choose **Heads** or **Tails**
* Random coin flip using `Math.random()`
* Instant result display
* Simple and interactive gameplay
* Beginner-friendly JavaScript project

## Built With

* HTML5
* JavaScript
